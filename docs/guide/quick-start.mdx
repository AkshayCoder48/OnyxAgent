title: Quick Install

---

Clone the repository, install dependencies, and start — it takes just a few commands.

Supports Linux, macOS, and Windows. Requires Python 3.7-3.13 (3.9 recommended).

## Install Command

Clone the repository and install dependencies:

```bash
git clone https://github.com/AkshayCoder48/OnyxAgent.git
cd OnyxAgent
pip install -r requirements.txt
cp config-template.json config.json
python app.py
```

The setup automatically performs these steps:

1. Check Python environment (requires Python 3.7+)
2. Clone project from GitHub
3. Install Python dependencies and Onyx CLI
4. Copy config template
5. Start service

By default, the Web console starts after installation. Access `http://localhost:9899` to begin chatting.

<Note>
  **Deploying on a server?** By default `web_host` only listens on `127.0.0.1` (local access only). Set `web_host` to `0.0.0.0` in `config.json` to make the console reachable from outside, and set `web_password` to protect it. Don't forget to open port `9899` in your firewall or security group — ideally restricted to specific IPs.
</Note>

## Management Commands

After installation, use the `onyx` command to manage the service:

| Command | Description |
| --- | --- |
| `onyx start` | Start service |
| `onyx stop` | Stop service |
| `onyx restart` | Restart service |
| `onyx status` | Check run status |
| `onyx logs` | View real-time logs |
| `onyx update` | Update code and restart |
| `onyx install-browser` | Install browser tool dependencies |

See the [Commands documentation](/cli/index) for more details.

<Note>
  If the `onyx` command is not available, you can use `./run.sh <command>` (Linux/macOS) or `.\scripts\run.ps1 <command>` (Windows) as a fallback. Both are functionally equivalent.
</Note>
